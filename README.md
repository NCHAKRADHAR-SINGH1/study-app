# Project Overview
This is a study/learning web application designed to help users improve their knowledge in various subjects.

# Setup Instructions
1. Clone the repository: `git clone https://github.com/NCHAKRADHAR-SINGH1/study-app.git`
2. Change directory: `cd study-app`
3. Install dependencies: `npm install`
4. Run the application: `npm start`

# Usage
- Navigate to `http://localhost:3000` in your web browser.
- Explore the different subjects available on the application.

# Scripts
- `npm start` - Starts the application.
- `npm test` - Runs the tests.

# Tech Stack
- Frontend: React.js
- Backend: Node.js
- Database: MongoDB
